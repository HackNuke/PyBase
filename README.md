PyBase is a DataBase Manager for multiple filetypes including SQLite3.

It's focused on the ease and effectiveness for the administration of databases.

------

The PyBase documentation is built using [VuePress](https://vuepress.vuejs.org/).

## Installation

```console
npm install
```

## Local Development

```console
npm run dev
```

> **This command starts a local development server.
> Most changes are reflected live without having to restart the server.**

## Build

```console
npm run build
```

> **This command generates static content into the `src/.vuepress/dist` directory and
> can be served using any static contents hosting service.**
